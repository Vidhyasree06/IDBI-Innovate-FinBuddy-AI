# FinBuddy AI 🤖💰
**An Avatar-Led Micro-Advisory Integration for Next-Gen Banking**

FinBuddy AI is an intelligent micro-advisory layer designed to integrate directly into mobile banking frameworks. It automatically scans transactional history to identify idle cash surpluses and leverages a conversational AI assistant to guide users toward smart, one-click asset diversification.

---

## 🚀 Key Features
* **Automated Idle-Cash Scanner:** Analyzes spending patterns to identify unutilized funds.
* **UPI Fractional Round-Ups:** Dynamically rounds daily transaction costs to the nearest ₹10 increment, routing the pocket change into secure micro-savings.
* **Conversational Avatar UI:** A friendly, responsive digital assistant that simplifies complex financial metrics into plain language.
* **Secure Data Execution:** Parses transactional details on-device to uphold strict bank-level security and data privacy parameters.

---

## 🛠️ Technology Stack
* **Backend Runtime:** Python, FastAPI, FastStream
* **AI/ML Intelligence:** Llama-3-8B-Instruct, LangChain
* **Avatar Interface:** Rive Vector Engine, WebGL
* **Storage Layer:** PostgreSQL, Redis Caching

---

## 📈 Automated Investment Strategy
FinBuddy AI optimizes wealth management by dividing surplus funds across four core pathways:
1. **50%** → IDBI Systematic Deposits (Reliable long-term baseline)
2. **25%** → Low-Risk Liquid Funds (Flexible emergency cash)
3. **15%** → Digital Gold Round-ups (Fractional inflation hedges)
4. **10%** → Micro-Equity Mutual Funds (Growth index creation)
